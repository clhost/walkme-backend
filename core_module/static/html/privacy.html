WalkMe. Политика конфиденциальности

  • Для пользования WalkMe пользователь должен авторизоваться через любую доступную социальную сеть в целях идентификации своей личности.
  • Из социальной сети в WalkMe сохраняется аватарка пользователя.
  • Данные, собираемые из социальной сети, недоступны для третьих лиц, в том числе для других пользователей WalkMe.
  • WalkMe не продает контент.
  • Безопасность Вашей персональной информации не будет нарушена.
  • WalkMe не отправляет на электронную почту рассылку новостей.
